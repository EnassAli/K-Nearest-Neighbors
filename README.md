# K-Nearest-Neighbors

# K-Nearest Neighbors (KNN) Classification Assignment

This project implements the K-Nearest Neighbors (KNN) algorithm using Python and Scikit-learn for classification. The assignment includes data exploration, feature standardization, model training, prediction, and performance evaluation.

## Tasks Completed
- Data exploration and visualization  
- Feature scaling using StandardScaler  
- Train-test split  
- KNN model implementation  
- Model evaluation using Confusion Matrix and Classification Report  
- Finding the optimal K value using the Elbow Method  
- Retraining the model with the best K value  

## Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

## Objective
The goal of this assignment is to apply KNN classification and evaluate how different K values affect model performance.
